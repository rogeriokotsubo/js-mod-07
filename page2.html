Exercício 2


Utilizando a mesma estrutura de 3 arquivos (HTML+CSS+JS):

Crie uma página web que contenha um formulário que contenha um campo do tipo CEP (input type text), considerando:

  1. Capturar o evento de input para não permitir ao usuário a inserção de um caractere que não seja numérico;
  2. Ao digitar o CEP, capturar o evento de keyup para poder preencher o campo até o limite de tamanho de um CEP não permitindo mais números do que o permitido e adicionando, se necessário o 'hífen' natural de um CEP, isto é, a medida que o usuário digita um número, o campo de CEP deve conter o valor do CEP (incluindo o hífen).
  3. Se o usuário apertar a tecla 'backspace' (keydown) ele deve apagar o último número inserido.
