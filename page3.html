Exercício 3


Utilizando a mesma estrutura de 3 arquivos (HTML+CSS+JS):

Crie uma página web que possua dois campos, um do tipo textarea, outro do tipo text e um botão de enviar, considerando:

  1. A página deve simular um chat, onde possibilite o envio de uma mensagem de forma que a mensagem digitada no campo do tipo text, ao clicar no botão enviar, deve aparecer no campo do tipo textarea (sem apagar o conteúdo do campo textarea adicionando uma linha a mais a cada inserção);
  2. O campo textarea deve ter uma largura e altura definida de forma a exibir aproximadamente as últimas 10 linhas e, caso possua mais linhas apareça uma barra de scroll para visualizar todas as mensagens que excedam este limite;
  3. Se o usuário digitar a mensagem e apertar a tecla 'enter', a mensagem deve ser enviada como se o usuário tivesse apertado o botão enviar;
  4. O campo textarea deve receber as mensagens e automaticamente fazer um scroll até a última mensagem permitindo que a última mensagem seja sempre visualizada (utilizar o scrollTop para isso);
  5. Adicionar um botão para poder 'limpar' o conteúdo do campo textarea que ao ser clicado apague o conteúdo do campo textarea.
